# Sorting Visualizer

An interactive web application that visually demonstrates how different sorting algorithms work.  
This tool helps users understand sorting techniques by displaying real-time animated bar visualizations of the sorting process.

---

## Features

- Interactive visualization of sorting algorithms
- Real-time animation of comparisons and swaps
- Multiple sorting algorithms supported
- Generate new random arrays
- Adjustable sorting speed
- Clean and modern dark UI

---

## Algorithms Implemented

- Bubble Sort
- Insertion Sort
- Selection Sort
- Merge Sort
- Quick Sort
- Radix Sort

---

## Tech Stack

- React.js
- JavaScript
- CSS
- Vite
